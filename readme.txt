生成口算题目的小软件

因为疫病流行，小学二年级的儿子在家呆了几个月，简直玩疯了，以前学过的东西也忘了不少。
为了让他复课以后不要落后太多，数学方面就给他练习练习口算吧。

软件使用QT做界面显示、操作和打印，在Visual Studio 2017 Community下编译通过。

试题生成
试题生成部分代码在Questions.h/.cpp中，用名称-生成函数对应的方式生成各种试题。
采用单例模式，保证生成部分的唯一性。

试卷结构
---试卷(Paper)包含若干部分(Section)，将顺序生成各部分的题目；
---部分(Section)包含若干集合(Set)，可以设置总数(Count)，可以分栏(Column)打印，也可以将本部分内所有集合的题目打乱顺序（Disorder）；
---集合(Set)包含若干同一类型的题目。当集合题目数量设置为0，且部分(Section)的总数设置不为0时，集合将参与均分没有分配的题目数量。


已知BUG
***当生成的试卷无法在一张纸上打印时，试题的打印位置会发生偏移；
***当生成的试卷在一张纸以内，也可能发生试题打印位置偏移的现象，且与当时的显示窗口(QTextEdit)大小相关。
